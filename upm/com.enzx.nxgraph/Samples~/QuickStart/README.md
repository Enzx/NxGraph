# Quick Start sample

This sample shows how to:

- build a tiny graph with `GraphBuilder`
- assign readable node names with `.SetName(...)`
- execute a `StateMachine` from a Unity `MonoBehaviour`

## Use

1. Import the sample from the Unity Package Manager.
2. Add `NxGraphQuickStartBehaviour` to a `GameObject`.
3. Enter Play Mode.
4. Inspect the Console output.
